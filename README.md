# Coupang Capital Allocation Case Study

An independent public-data strategy case study evaluating Coupang's 2026-2028 capital allocation options across Product Commerce, Taiwan, Eats/WOW, and Farfetch.

## Disclaimer

This is an independent portfolio case study based only on publicly available information. It is not affiliated with, sponsored by, or endorsed by Coupang. It is not investment advice and should not be interpreted as client work.

## Executive Takeaway

Coupang should run a disciplined growth portfolio in 2026-2028:

1. Protect and restore Product Commerce margin as the funding engine.
2. Selectively scale Taiwan only if unit economics improve.
3. Keep Eats/WOW investment tied to measurable incremental LTV.
4. Constrain Farfetch until standalone turnaround evidence improves.

## Why This Case

Coupang's Q1 2026 results created a useful strategy question: how should a company balance a profitable core with expensive growth options when consolidated margin capacity tightens?

The case uses public filings and investor materials to build a consultant-style decision package:

- Problem framing
- Financial baseline
- Initiative evidence map
- Weighted option scorecard
- Final recommendation memo
- Board-style deck outline

## Method Note

The scorecard is a decision tool, not a full valuation model. Public filings do not disclose standalone unit economics for Taiwan, Eats/WOW, or Farfetch. The analysis therefore separates official facts, management commentary, calculations, and directional inferences.

## Case Quality

This is a strong public-data practice case. It is not equivalent to a live consulting project because it lacks internal unit economics, stakeholder interviews, and a full 2026-2028 cash-flow model.
